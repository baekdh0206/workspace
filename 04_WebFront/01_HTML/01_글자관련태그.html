기타 글자 관련 태그

인용문

b 태그 : 글자를 굵게 표시하는 태그
strong 태그 : 굵게 강조하는 태그
i 태그 : 글자를 이탤릭체(기울인 글자)로 표시하는 태그
em 태그 : 이탤릭체로 강조하는 태그
abbr 태그 : 마우스 오버 시 툴팁 형식으로 title 내용을 출력
cite 태그 : 웹 문서나 포스터에서 참고 내용을 표시하는 태그

내가 경험한 가장 흥미진진한 일은 누군가를 만나는 일이다 - 영화, '비포선셋' 중

code 태그 : 컴퓨터 인식을 위한 코드를 작성하는 태그

    function saveTheLocal(){
        console.log("저장");
    }
small 태그 : 글자를 작게
sup 태그(위 첨자) : x2
sub 태그(아래 첨자) : log3
