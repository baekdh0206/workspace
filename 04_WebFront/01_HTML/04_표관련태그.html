04_표 관련 태그

        
table 태그

        - 표를 작성할 수 있는 판
    
        
tr 태그

        - Table Row의 약자로 한 행을 나타내는 태그
    
        
td 태그

        - Table Data의 약자로
        각 행의 1칸(컬럼, 열)을 나타내는 태그
    
        
th 태그

        - Table Header의 약자로
        컬럼명이나 중요한 데이터를 표시할 때 사용

        - td와 기본 성질은 같으나
        출력되는 글자가 굵은 글씨, 가운데 정렬됨
    
        
caption 태그

        - 테이블의 제목이나 설명을 추가하는 태그

행 병합(rowspan), 열 병합(colspan)

        - td 또는 th에만 작성 가능한 속성

        - rowspan : 행 끼리 병합(상하 병합)
        
        - colspan : 열 끼리 병합(좌우 병합)
