영역을 나누는 형식(성질)

        
block 형식

        - 공간을 수직 분할
        -> 수직으로 되어있는 화면을 가로로 잘라서
            여러 행을 만드는 것
        
        - 크기를 지정하는 css 속성인 width, height를 사용할 수 있다!

        ex) h1~h6 (제목 영역을 분할)
            p/pre (문단 분할)
            hr (수평선을 그어 화면 분할)
            div (영역 분할)
            시맨틱 태그(div 태그에 이름을 넣은 태그)
                -> header, main, footer, section, article, aside
    
        
inline 형식

        - 공간을 수평 분할
        -> 수평으로 되어있는 화면을 세로로 잘라서
            여러 열(컬럼)을 만드는 것
        
        - 크기를 지정하는 css 속성인 width, height를 사용할 수 없다...    
        
        ex) b/strong, i/em, mark, span, img

block / inline 차이점 1 : 줄 바꿈

div 태그(block)

첫 번째 영역
두 번째 영역
세 번째 영역

span 태그(inline)

첫 번째 영역 두 번째 영역 세 번째 영역 
        block 형식은
        한 줄을 모두 차지하기 때문에
        뒤에 작성된 내용이 다음 줄에 출력된다.

        inline 형식은
        작성된 내용의 크기만큼 영역을 차지하기 때문에
        뒤에 작성된 내용이 줄을 바꾸지 않고 옆에 붙어서 출력된다.

block / inline 차이점 2 : 영역 지정 방식

  • block : 사각형 박스 형태로 영역을 지정
  • inline : 내부에 작성된 내용(content) 단위로 영역을 지정

  • block : 크기 지정 가능(O)
  • inline : 크기 지정 불가능(X)

        - 웹 문서 내부에
        또 다른 웹 문서를 추가하는 태그(inline 형식)
