하이퍼링크 관련 태그 : a(anchor) 태그

        a태그는 현재 웹 문서와 다른 웹문서를 연결해서
        페이지간의 이동을 가능하게 만드는 기능을 제공

        -> 이 때 a태그로 감싸진 문자열 == 하이퍼링크

        * 하이퍼링크는 웹 문서(HTML)가 
        다른 문서와 구분되는 핵심적인 기능.

        텍스트 또는 이미지를 클릭해서 페이지 이동이 가능하다
        + 페이지 내부 이동(점프)도 가능
