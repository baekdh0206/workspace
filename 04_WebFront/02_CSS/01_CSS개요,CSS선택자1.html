CSS(Cascading Style Sheets)

        마크업 언어가(HTML) 실제로 화면에 표시되는 방법
        (모양, 색상, 크기 등...)을 기술하는 언어.

        W3C(웹 표준 지정 기관)에서 CSS를 표준으로 지정.

        부모 요소를 대상으로 하는 CSS 코드는 자식 요소로 상속되며

        하나의 요소를 대상으로 두 가지 이상의 CSS 코드가 작성되면
        작성 순서, 선택자에 따라 우선순위가 결정된다.

선택자(Selector)란?

        현재 HTML 문서 내에서 특정한 요소를 선택하는 문자 또는 문자열.

태그 선택자

        현재 HTML 문서 내에 같은 태그를 선택하는 선택자

        태그명 { CSS 코드; }
strong 태그에 작성된 내용입니다
span 태그에 작성된 내용입니다
  • strong 태그에 작성된 내용입니다
  • span 태그에 작성된 내용입니다

id 선택자

        - HTML 요소 중 id 속성 값이 일치하는 요소를 선택하는 선택자

        (중요!)
        id 속성 값은 페이지 내에서 중복되는 값이 있어서는 안된다!
        (== 유일한 값을 지녀야 한다)

        #id속성값 { CSS 코드; }

        # 기호 == id 

        id 속성 값은 카멜표기법으로 작성해야 한다!

class 선택자(class 속성 값)

        - HTML 문서 내에서 같은 class 속성 값을 가지는
            요소를 모두 선택하는 선택자.

        - class(종류, 분류) 속성 값을 이용해서
            관련있는 요소끼리 그룹을 만들 수 있다.

        .class속성값 { CSS 코드; }

        . 기호 == class

        - class 속성 값은 하이픈(-) 표기법을 사용
