기본 속성 선택자

        태그에 작성된 특정 속성을 선택하는 선택자

        선택자[속성명="속성값"] { CSS 코드; }

        * "", '' 둘 다(문자열) 사용 가능

        (참고)
        1) 선택자는 생략할 수 있다.
            -> 생략 시 HTML 문서 내 속성이 일치하는 모든 요소 선택

        2) "속성값" 에서 양쪽 쌍따옴표("")는
            홑따옴표('') 로 변경하거나 생략도 가능하다.

자식 선택자 (>) 

        지정된 요소 바로 하위(1단계 아래)에 존재하는 
        요소를 모두 선택하는 선택자

        선택자1 > 선택자2{ CSS 코드; }

        - 선택자1 : 부모 요소 선택자
        - 선택자2 : 자식 요소 선택자

반응 선택자 (:가상 클래스)

        사용자의 움직임에 반응하여 스타일이 달라지는 선택자

        - 요소를 클릭하고 있는 경우  (:active)
        - 요소에 마우스가 올라가 있는 경우  (:hover)
        - 요소에 초점이 맞춰졌을 때  (:focus)

상태 선택자

        입력 양식(input, input 관련된 태그)의
        상태에 따라 선택되는 선택자.

        :checked -> checkbox, radio가 체크 되었을 때 선택

        :enabled / :disabled -> 요소가 사용 가능한/불가능한 상태일 때 선택

동위 선택자

        동위 관계(동등한 위치에 있는 관계 == 같은 레벨 == 형제 관계)에서 
        현재 요소 다음에 위치한 요소를 선택하는 선택자

        A
        B (1)(2)
        B (2)

        (1) A 바로 뒤(다음)에 있는 B를 선택 ->  + 기호
        A선택자 + B선택자 { CSS코드; }

        (2) A 뒤에 있는 모든 B를 선택 ->  ~(틸드) 기호
        A선택자 ~ B선택자 { CSS코드; }
