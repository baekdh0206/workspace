CSS 선택자의 우선순위

        HTML, CSS 코드는 기본적으로 작성된 순서(위->아래/ 왼쪽->오른쪽)대로
        해석이 진행된다. (== 랜더링)

        단 CSS 코드는 여러 선택자를 이용해서 
        하나의 요소를 선택하는게 가능하기 때문에 
        작성 순서 / 선택자의 우선 순위에 따라 적용되는 CSS가 결정된다.

        * 우선 순위 설정 전 알아둬야 할 것
        1) 동일한 우선 순위로 CSS 코드가 작성된 경우
            마지막으로 작성된 코드가 적용된다.

            ex) #test1{ color:red; }
                #test1{ color:blue; } -> 마지막이 적용

        2) 여러 선택자를 이용해서 같은 요소를 선택하더라도
            설정하는 CSS 속성이 다르면 모두 반영된다.

            ex) #test2 { color : white; }
                #test2 { background-color : red; }
                #test2 { font-size : 30px; }
                -> 모두 적용

        1순위 : !important;
        2순위 : inline-style (요소에 style 속성 추가)
        3순위 : 아이디 선택자
        4순위 : 클래스 선택자
        5순위 : 태그 선택자
