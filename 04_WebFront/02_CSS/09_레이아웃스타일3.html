배치 관련 스타일(position)

        postion은 요소의 위치를 지정하는 속성

        - relative (상대적인)
        지정된 요소 내부에 다른 요소가 배치될 때
        상대적인 위치를 지정할 수 있도록 기준을 설정하는 속성
        (기준점 만들기!)
            + 내부에 작성된 요소에는
                상대적 위치를 지정하기 위한 
                top, bottom, left, right 속성을 사용할 수 있음.

        - absolute (절대적인)
        기본 요소의 배치 순서를 무시하고
        지정된 절대 위치에 요소를 배치함.
        (원하는 곳에 마음대로 배치)

        - fixed (고정된)
        항상 고정된 위치에 요소를 배치함.
        (화면이 스크롤되도 항상 같은 위치,
        브라우저를 기준으로 배치됨)
        
        - sticky 
        처음에는 일반적인 요소였다가
        화면에서 벗어나는 경우
        지정된 위치에 붙어서 고정
