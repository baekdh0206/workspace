글자 색을 지정하는 속성 : color

        color에 작성할 수 있는 값의 종류

        - 색상명(영문)
        - rgb(0~255, 0~255, 0~255) red,green,blue
        - rgba(0~255, 0~255 ,0~255 ,0~1) red,green,blue, 투명도
            0: 투명   1: 불투명
        - 16진수 표기법 #RGB (#fff, #ffffff)
        - hsl(색조, 채도, 명도(밝기))
        - hsla(색조, 채도, 명도(밝기), 투명도)
  • 색상명
  • rgb
  • rgba
  • 16진수 3자리
  • 16진수 6자리
  • hsl
  • hsla

글자 줄을 그어 꾸미는 속성 : text-decoration

        윗줄   : overline
        중간줄 : line-through
        아랫줄 : underline
        없애기 : none
  • overline
  • line-through
  • underline
  • none

글꼴의 굵기 : font-weight

        - 숫자(100(얇음) ~ 900(두꺼움))
        - normal : 폰트에 기본으로 지정된 두께(300~400)
        - bold : 굵게(700)

        - bolder : 상속 받은 굵기보다 굵게
        - lighter : 상속 받은 굵기보다 얇게

글꼴 크기 : font-size

        - 고정 크기 단위(px, pt)
        - 가변 크기 단위(em, rem, %)
            -> em  : 부모 글꼴 크기 비례
            -> rem : 최상위 부모 글꼴 크기 비례

font-family : 글꼴 지정 속성
