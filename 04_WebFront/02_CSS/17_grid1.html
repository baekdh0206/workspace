display : grid

        행(row)과 열(column)을 이용하여
        2차원적으로 화면에 요소를 배치할 수 있게하는 형식

        grid layout을 이용할 때 반드시 알아야 하는 것!

        1. grid의 구성
        - grid container : grid를 지정하고자 하는 요소를 감싸는 요소(부모)
        - grid cell : grid container 내부 요소(배치되는 자식)

        (grid container와 grid cell에 사용되는 CSS 속성이 나누어져 있다)

        2. grid는 행과 열의 개수 + 크기를 지정해야 한다!

        grid container 전용 속성

        1. display : grid
        - 감싸는 요소를 gird 형식으로 변경
            -> 내부에 작성된 요소를 grid로 배치하겠다는 의미

        2. grid-template-rows : 행의 개수와 각 행의 크기 지정
        3. grid-template-columns : 컬럼의 개수와 각 컬럼의 크기 지정
        - grid-template-rows/columns에 작성하는 값의 개수 만큼 행/열이 생성됨.
        - 작성하는 값은 각 행/열의 크기로 지정됨

        4. repeat(횟수, 크기) 함수
        - 똑같은 값이 여러 번 반복되는 경우 이를 간단히 작성할 수 있는 함수

        5. 크기 지정 단위 fr(fraction : 나누기)
        - 감싸고 있는 부모(gird container) 크기를 일정 비율로 나눠
            행/열의 크기를 지정하는 단위

        6. minmax(최소, 최대) 함수
        - 지정된 행/열에 작성된 내용이
        행/열의 높이 보다 작으면 min
        행/열의 높이보다 크면 max 값을 적용하는 함수
        
        - 값으로 auto를 지정할 경우 content의 크기 만큼으로 조정됨

        7. auto-fit / auto-fill 속성 값
        - 화면/부모 요소의 크기에 따라 컬럼의 개수를 자동으로 조절하는 속성 값

        minmax()와 함께 사용 시 차이점이 생김
        auto-fit : 컬럼 개수 조정 후 남는 공간이 있으면 cell을 늘려서 꽉 채움
        auto-fill :  컬럼 개수 조정 후 남는 공간이 있어도 지정된 컬럼 크기만큼만 표시


        8. grid-gap : 행/열 사이의 간격을 지정
        + grid-row-gap : 행 사이 간격을 지정
        + grid-column-gap : 열 사이 간격을 지정

        * grid-gap, grid-row-gap, grid-column-gap
        3가지 속성은 css 코드 작성시 경고 발생
        -> 최신 브라우저에서 모두 gap, row-gap, column-gap으로 대체되어 
            더 이상 사용하지 말라는 경고

        (참고) flex 정렬 관련 속성 값 중
        flex-start, flex-end는 
        display : flex; 가 적용된 요소에서만 작성 가능 하지만

        flex 뿐만 아니라 block 형식 요소, table, grid, flex에
        시작, 끝이라는 공통된 정렬 방법을 제공하기 위해
        start, end라는 속성 값을 제공 중
