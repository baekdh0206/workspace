innerHTML

        JS에서 요소 전체를 읽어들이거나 변경하는 속성

        - 내용에 포함된 요소 자체를 읽어옴(태그, 속성, 내용)
        - 내용 변경 시 포함된 태그, 속성이 모두 해석된다

innerHTML 응용

[window.]alert("내용")

        - alert : 경고, 경보, 알리다
        - 브라우저에 경고창(알림창, 대화 상자)을 띄우는 함수

[window.]confirm("내용")

        - "내용"에 대한 예/아니오 결과를 얻고자 할 때 사용하는 함수
        
        - 대화 상자에 예/아니오 (또는 확인/취소) 버튼이 생기고
            예(확인) 클릭 시 true
            아니오(취소) 클릭 시 false 반환

[window.]prompt("내용")

        - 내용을 입력할 수 있는 대화 상자

        - 내용 입력 후 확인 클릭
            -> 입력한 내용이 그대로 반환

        - 취소 클릭
            -> null 반환
